## Introduction

GeoLib est une plateforme web qui facilite l'accès aux travaux académiques (TFC, mémoires, thèses, articles, rapports de stage) associés à des zones géographiques spécifiques. Grâce à notre système de cartographie intégré, vous pouvez découvrir des documents liés à des régions particulières du monde.

### Rôles utilisateur

- **Lecteur** : Accès en lecture aux documents, recherche et emprunt
- **Auteur** : Publication de documents et gestion de profil
- **Bibliothécaire** : Gestion du catalogue et des emprunts
- **Administrateur** : Accès complet à toutes les fonctionnalités

## Premiers pas

### Inscription

1. Cliquez sur "S'inscrire" dans le menu principal
2. Remplissez le formulaire avec vos informations :
   - Nom d'utilisateur
   - Email
   - Mot de passe
3. Sélectionnez votre rôle (si applicable)
4. Cliquez sur "Créer un compte"

### Connexion

1. Cliquez sur "Se connecter"
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"

### Réinitialisation du mot de passe

Si vous avez oublié votre mot de passe :
1. Cliquez sur "Mot de passe oublié"
2. Entrez votre adresse email
3. Suivez les instructions envoyées par email

## Pour les Lecteurs

### Recherche de documents

1. Utilisez la barre de recherche principale pour rechercher par titre ou mots-clés
2. Appliquez des filtres avancés :
   - **Type de document** : TFC, Mémoire, Thèse, Article, etc.
   - **Année académique** : Sélectionnez une ou plusieurs années
   - **Zone géographique** : Utilisez la carte pour définir une région
   - **Faculté/Département** : Filtrez par institution

### Visualisation sur la carte

1. Accédez à la section "Carte" ou "Zones d'étude"
2. Les documents sont représentés par des marqueurs sur la carte
3. Cliquez sur un marqueur pour voir les détails du document
4. Utilisez les contrôles de zoom pour explorer différentes échelles

### Consultation d'un document

1. Cliquez sur un document dans les résultats de recherche
2. Lisez le résumé et les métadonnées
3. Si disponible, téléchargez le document complet (PDF, etc.)
4. Consultez les informations géographiques associées

### Emprunt de documents

1. Vérifiez la disponibilité du document
2. Cliquez sur "Emprunter"
3. Sélectionnez la durée d'emprunt (si applicable)
4. Confirmez l'emprunt
5. Le document sera ajouté à votre historique d'emprunts

### Gestion de votre compte lecteur

- **Historique d'emprunts** : Consultez vos emprunts passés et actuels
- **Préférences** : Personnalisez vos filtres de recherche par défaut
- **Notifications** : Gérez vos alertes pour les retours et nouvelles publications

## Pour les Auteurs

### Publication d'un document

1. Accédez à votre tableau de bord auteur
2. Cliquez sur "Nouveau document"
3. Remplissez les informations :
   - Titre
   - Résumé
   - Type de document
   - Année académique
   - Faculté et département
4. Associez des zones géographiques :
   - Utilisez l'éditeur de carte intégré
   - Dessinez des points, polygones ou lignes
   - Importez des fichiers GeoJSON
5. Téléversez le fichier du document
6. Cliquez sur "Publier"

### Gestion de votre profil

1. Accédez à "Mon profil"
2. Ajoutez une biographie
3. Téléversez une photo de profil
4. Gérez vos publications

### Modification de documents

1. Dans votre tableau de bord, cliquez sur un document existant
2. Apportez les modifications nécessaires
3. Sauvegardez les changements

## Pour les Bibliothécaires

### Gestion du catalogue

1. Accédez au tableau de bord bibliothécaire
2. **Ajouter des documents** :
   - Importez des documents en masse
   - Validez les soumissions d'auteurs
3. **Modifier les métadonnées** :
   - Corrigez les informations
   - Ajoutez des mots-clés
4. **Gérer les catégories** :
   - Créez de nouvelles catégories
   - Organisez les documents

### Gestion des emprunts

1. Consultez la liste des emprunts en cours
2. **Valider les emprunts** :
   - Approuvez les demandes d'emprunt
   - Gérez les files d'attente
3. **Traiter les retours** :
   - Marquez les documents comme retournés
   - Gérez les retards et pénalités
4. **Historique** : Consultez l'historique complet des emprunts

### Gestion des utilisateurs

1. Accédez à la section "Utilisateurs"
2. **Valider les inscriptions** : Approuvez les nouveaux comptes
3. **Gérer les rôles** : Modifiez les rôles des utilisateurs
4. **Bloquer/Activer des comptes** : Gérez l'accès des utilisateurs

## Pour les Administrateurs

1. **Paramètres généraux** :
   - Configurez les limites d'emprunt
   - Gérez les délais de retour
2. **Gestion des paiements** :
   - Configurez les tarifs d'abonnement
   - Gérez les paiements et factures
3. **Maintenance** :
   - Sauvegardez la base de données
   - Gérez les mises à jour système

### Statistiques et rapports

1. Accédez au tableau de bord d'administration
2. Consultez les statistiques d'utilisation :
   - Nombre de documents
   - Taux d'emprunt
   - Activité des utilisateurs
3. Générez des rapports personnalisés

## Dépannage

### Problèmes courants

#### Je ne peux pas me connecter
- Vérifiez votre email et mot de passe
- Assurez-vous que votre compte est activé
- Essayez de réinitialiser votre mot de passe

#### La recherche ne donne pas de résultats
- Vérifiez l'orthographe des termes de recherche
- Essayez des termes plus généraux
- Vérifiez vos filtres actifs

#### Problème avec la carte
- Actualisez la page
- Vérifiez votre connexion internet
- Essayez un autre navigateur

#### Erreur lors du téléversement
- Vérifiez le format du fichier
- Assurez-vous que la taille ne dépasse pas la limite
- Réessayez avec un fichier plus petit

### Support

1. Consultez cette documentation
2. Vérifiez les mises à jour de l'application
3. Contactez l'équipe de support via le formulaire de contact
4. Ouvrez un ticket sur notre plateforme de support
